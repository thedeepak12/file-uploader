# File Uploader

A comprehensive cloud storage solution that enables users to securely store, organize, and manage their files through an intuitive web interface. Users can create personalized folders, upload various file types, and maintain complete control over their digital assets with full CRUD functionality.

## Features

- **User Authentication**: Secure session-based authentication.
- **File Management**: Upload, download, and organize files with ease.
- **Folder Organization**: Create, read, update, and delete folders for better file organization.
- **Cloud Storage**: Store files securely in Cloud.

## Tech Stack

- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **TypeScript** - Type-safe JavaScript
- **Prisma** - Database ORM
- **Passport.js** - Authentication
- **Multer** - File upload middleware
- **EJS** - Templating engine

## What I Learned

- **Prisma**: Set up database schema with User, Folder, and File models with proper relationships.
- **File Upload**: Integrated Multer middleware for handling file uploads and processing.
- **File Management**: Built complete CRUD operations for folders and files with proper authorization checks.
